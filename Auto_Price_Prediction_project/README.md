# **Auto Price Prediction**

This project focuses on developing a machine learning model to predict automobile prices based on various features such as brand, engine specifications, mileage, and other vehicle attributes. The goal is to help buyers and sellers make informed pricing decisions by leveraging data-driven insights.

The dataset was preprocessed by handling missing values, encoding categorical variables, and performing feature engineering to improve model performance. Exploratory Data Analysis (EDA) was conducted to understand the key factors influencing car prices and identify important trends and relationships.

Multiple regression algorithms such as Linear Regression, Random Forest, and other advanced models were implemented and compared. Model performance was evaluated using metrics like R-squared, Mean Absolute Error (MAE), and Root Mean Squared Error (RMSE), with hyperparameter tuning applied to enhance prediction accuracy.

The final model provides reliable price predictions and highlights the most significant features affecting automobile valuation. This project demonstrates end-to-end machine learning capabilities, from data preprocessing and model building to evaluation and business insight generation.
