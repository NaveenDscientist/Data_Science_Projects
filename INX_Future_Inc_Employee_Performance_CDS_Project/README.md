# **Employee Performance Analysis & Prediction**

This project focuses on analyzing employee data and building a machine learning model to predict performance levels, helping organizations make data-driven HR decisions. The goal is to identify key factors influencing employee performance and improve workforce productivity and management strategies.

The dataset was preprocessed by removing duplicates, handling outliers, and performing feature engineering to create meaningful variables such as Experience Ratio and Total Satisfaction. Exploratory Data Analysis (EDA) was conducted to uncover patterns, revealing strong relationships between performance and factors like salary hike, job satisfaction, and work-life balance.

Multiple classification models including Logistic Regression, Decision Tree, Random Forest, and Gradient Boosting were implemented and optimized using GridSearchCV and cross-validation. Class imbalance was handled using SMOTE, and model performance was evaluated using Accuracy, Confusion Matrix, and Classification Report, with ensemble models achieving the best results.

The final model provides actionable insights into employee performance drivers and supports better decision-making in hiring, compensation, and employee engagement. This project demonstrates end-to-end data science expertise, from data analysis and feature engineering to predictive modeling and business recommendations.
